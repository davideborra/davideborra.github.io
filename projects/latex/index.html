Beamer
This is my LaTeX preamble, constantly updated with new macros and features, necessary to compile everyone of my documents. It is compatible with article, book and beamer classes, though it might give some problems while used in beamer mode: in this case delete either remove from LambdaTeX the theorem definitions or enable the notheorems option of beamer. To load it, simply include it in your LaTeX directory and call it via \usepackage{lambdatex}.

Book
This is an extension of LambdaTeX meant to be used in books or articles. It mainly contains graphic enviroments for theorems, remarks, etc. To load it, simply include it in your LaTeX directory and call it via \usepackage{lambdabook}. It comes with some useful options:
  • colorful enables coloured environments,
  • bw disables coloured environments,
  • print optimises the document for printers.
Note: in order to work, it needs one between colorful and bw to be enabled.
