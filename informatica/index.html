Lambdalib.h

La nostra libreria per il C, con alcune funzioni utili che è possibile richiamare in modo semplice e veloce. La libreria è disponibile per il download al seguente link:

lambdalib - update 12-11-2022

Per utilizzarla è possibile seguire due procedimenti:

Tenendo la libreria nella stessa directory del sorgente

In questo caso la libreria non va richiamata tramite la classica sintassi con le parentesi angolari ma esse vanno sostituite dagli apostrofi.
#include "lambdalib.h"

Spostando la libreria nella directory del compilatore

In questo caso è necessario spostare la libreria nella directory dove poi il compilatore andrà a recuperarla (i percorsi sono riferiti alla configurazione di default fornita dai tool di installazione in un sistema operativo Windows 10:
  • Dev-C++: C:\Program Files (x86)\Dev-Cpp\MinGW64\x86_64-w64-mingw32\include
  • MinGW: C:\MinGW\include
In questo caso per l'utilizzo della libreria si utilizza la sintassi solita con le parentesi angolari:
#include <lambdalib.h>
